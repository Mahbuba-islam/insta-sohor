What does javaScript Event-loop do?

The event loop is the javaScript's asynchronous programming. An event loop is often the main loop in a program that typically waits for the user to trigger something. It has call stack and the call-back queue. If the call stack is empty , the event loop continuously checks the call stack to see if there's any function that's needs to run and will take the frist event from the queue and push it to the call stack. Once the stack is clear, the event loop triggers and checks the task queue for available callbacks. Each event is just a function callBack. It is resposible for collecting and executing the code and executing queued sub-tasks. The event loop performs internal opperations of any callbacks.
